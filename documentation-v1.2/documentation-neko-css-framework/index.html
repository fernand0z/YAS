Neko CSS Framework

Documentation by "LITTLE NEKO"

A CSS framework based on bootstrap 3

The Neko CSS Framework is an extension of the Twitter Bootstrap Framework. It gives you new tools to create your layouts, managing colors and useful components

Before you get started, please be sure to always check out these documentation files. We outline all kinds of good information, and provide you with all the details you need to use.

We tried to include anything you need in this documentation file. However, if you need help, please register to our support forum.
We will do our best to assist you with any issues/bugs related to the use of our themes and templates.
Theme support is offered to verified customers by forum only. Support requests made by email, tweet or any other method will be redirected back to this forum.
Unfortunately we are not able to provide support for customizations or using 3rd party plugins.

Components

All Bootstrap components are supported, please refer to Bootstrap documentation for grid information or complementary informations about components. http://getbootstrap.com.

On top of Bootsrap, we added a lot of components and utilities described in this document.

Semantic

As far as possible, we add more semantic to Bootstrap, in order to make the framework more readable and user friendly. Of course, if you are used to Bootstrap conventions, you can still use them.

Example with a button

My button

Bootstrap syntax

<a href="#" class="btn btn-default btn-lg">My button</a>

Neko Framework syntax

<a href="#" class="btn default large">My button</a>
